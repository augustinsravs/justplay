# 🎮 justplay

An interactive gaming platform built with **FastAPI** and **Modern Web Technologies**, featuring AI-powered games and real-time emotion detection.

## 🌟 Features

### 🎯 **Core Games**
- **Smart Tic Tac Toe**: Challenge our intelligent AI or play with friends
- **Emotion AI Scanner**: Real-time facial sentiment analysis using TensorFlow.js
- **AI Math Master**: Adaptive mathematical challenge with progressive difficulty
- **AI Trivia Challenge**: Dynamic trivia questions with AI-curated content

### 🔧 **Technical Features**
- **Real-time Interactions**: WebSocket support for live gameplay
- **Responsive Design**: Works seamlessly on desktop and mobile devices
- **Modern UI/UX**: Glassmorphism design with smooth animations
- **AI-Powered**: Integrated machine learning for emotion detection
- **Cross-Platform**: Runs on any device with a modern web browser

## 🛠️ Tech Stack

### **Backend**
- **[FastAPI](https://fastapi.tiangolo.com/)** - Modern Python web framework
- **[Uvicorn](https://www.uvicorn.org/)** - ASGI server implementation
- **[Python](https://python.org/)** - Core programming language

### **Frontend**
- **HTML5** - Modern semantic markup
- **CSS3** - Advanced styling with animations
- **JavaScript (ES6+)** - Interactive functionality
- **[TensorFlow.js](https://www.tensorflow.org/js)** - Machine learning in browser

### **AI/ML Integration**
- **MediaPipe Face Mesh** - Real-time facial landmark detection
- **Custom Emotion Detection** - AI-powered sentiment analysis
- **Adaptive Algorithms** - Dynamic difficulty adjustment
